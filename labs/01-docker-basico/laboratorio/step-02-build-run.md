# Step 02 - Build y ejecución

## Tarea

Crear `requirements.txt`, `main.py` y `Dockerfile` como en capítulos y ejecutar:

```bash
cd labs/01-docker-basico/trabajo/mi-analitica
docker build -t mi-analitica:1.0 .
docker run --rm mi-analitica:1.0
```

## Validación

La salida muestra el DataFrame sin errores.
