# Step 01 - Ejecutar y listar contenedores

## Tareas

```bash
docker run hello-world
docker run --name hello-world hello-world
docker ps
docker ps -a
docker run --rm hello-world
docker ps -a
```

## Validación

- Explica diferencia entre `ps` y `ps -a`.
- Explica qué hace `--rm`.
